Article Headline One

This is a rich-text content block styled by the .prose wrapper. It demonstrates how long-form content — blog posts, documentation, case studies — will render inside the design system.

A Second-Level Heading

Good typography makes content easy to scan. Each heading level has balanced spacing above and below, and body text has a comfortable line height for reading.

  • Unordered list items display with disc markers
  • Spacing between items keeps them scannable
  • Nested content inherits the system's tokens

A Third-Level Heading

Block quotes pull attention to testimonials, callouts, or cited passages:

The best design systems are invisible. They make every page feel cohesive without anyone noticing the infrastructure underneath.
  1. Ordered list items use decimal numbering
  2. They share the same spacing rhythm

Inline code spans are styled with a subtle background and monospace font. The prose wrapper ensures all descendant elements inherit a consistent look without extra classes.
